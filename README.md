Once you install the software, look for the clawpdf icon on your desktop. Double-click the icon to open the main window. You will see a clean workspace. ## 🛠️ Using the Features

You can perform common PDF tasks with ease.

*   **View Files:** Open your documents to read them. You can zoom in on pages or rotate them if they appear sideways.
*   **Merge Files:** Select multiple PDF files and drag them into the workspace together. The software combines them into one document. 
*   **Split Files:** Choose a range of pages to save as a new, separate file.
*   **Reduce Size:** If your file is too large to email, use the compress feature to shrink your file size while keeping text clear.
*   **Convert Formats:** Change PDF pages into image files like JPG or PNG if you need to share them on social media or websites.

This project uses WebAssembly. This technology allows browsers to run high-speed code. We use PDFium, an engine that handles PDF rendering. Because we combine these, the software runs fast and stays light on your system resources.
